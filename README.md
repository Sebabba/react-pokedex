## Quick Start

### Prerequisites

- Node.js 20+
- npm or yarn
- Modern browser

### Installation

**Clone the repository**

```bash
git clone https://github.com/Sebabba/react-pokedex.git
```

### Install Dependencies

```bash
cd react-pokedex && npm install
```

### Testing
```bash
npm test
```

### Production build
```bash
npm run build
```
